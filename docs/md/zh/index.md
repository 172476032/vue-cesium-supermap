## 介绍

<p style="text-indent:2em;">SuperMap iClient 3D for WebGL 一款在服务式 GIS 架构体系中， 无任何插件，跨浏览器的客户端产品。它基于Cesium开源框架,面向 HTML 5 的三维应用开发，快速构建内容丰富、响应迅速、体验流畅的三维真空间应用。支持加载：</p>

- SuperMap iServer 服务
- 地图服务
- 数据服务
- 量算服务
- 查询服务
- 空间关系服务
- 专题图服务
- 空间分析服务类
- SuperMap 云服务
- OGC 标准的服务 (WMS、WFS、WMTS、KML)
- 其他第三方服务，如天地图等
